# rxjs-range-slider

Keyboard-driven range slider using RxJS.

## Setup

```sh
yarn
```

Then do start and watch for changes

```
yarn run start
```

To build once

```
yarn run build
```

### Running tests

To run tests once

```sh
yarn run test
```

To run tests and watch for changes

```sh
yarn run test:watch
```
